# Boutique Villa Booking Platform

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your credentials.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- Supabase (Database & Auth)
- Stripe (Payments)
